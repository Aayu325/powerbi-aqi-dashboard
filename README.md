# 📊 AQI Power BI Dashboard

## 📌 Overview
This project analyzes Air Quality Index (AQI) data using Power BI.

## 🚀 Features
- AQI categorization (Good → Hazardous)
- Interactive dashboard with filters
- KPI cards and visual insights

## 🛠 Tools Used
- Power BI
- DAX
- Power Query
